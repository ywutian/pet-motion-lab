# 📋 步骤选择器使用指南

## 🎯 **功能说明**

**步骤选择器**允许你**直接跳转到任意步骤**，无需按顺序执行。

---

## 🚀 **如何使用**

### **1. 打开步骤选择器**

在主页（可灵AI页面）点击右上角的 **网格图标** 📱

```
┌─────────────────────────────────┐
│  🎬 可灵AI宠物动画生成           │
│                    [📱] [分步模式] │
└─────────────────────────────────┘
```

### **2. 填写宠物信息**

```
┌─────────────────────────────────┐
│  宠物信息                        │
│                                 │
│  Pet ID: pet_1234567890         │
│  物种: 猫                        │
│  品种: 布偶猫                    │
│  颜色: 蓝色                      │
└─────────────────────────────────┘
```

**说明：**
- **Pet ID**：步骤2-6需要（从步骤0获取）
- **物种**：猫/狗
- **品种**：例如"布偶猫"、"金毛"
- **颜色**：例如"蓝色"、"金色"

### **3. 上传图片（可选）**

如果你已经有处理好的图片，可以直接上传：

```
┌─────────────────────────────────┐
│  📤 上传图片（可选）             │
│                                 │
│  [上传去背景图片（步骤2用）]     │
│  [上传坐姿图片（步骤3用）]       │
└─────────────────────────────────┘
```

**说明：**
- **步骤2**：需要上传去背景的宠物图片
- **步骤3**：需要上传标准坐姿图片
- **其他步骤**：不需要上传图片

### **4. 选择步骤**

点击任意步骤卡片即可跳转：

```
┌─────────────────────────────────┐

┌─────────────────────────────────┐
```

## 📝 **使用场景**

### **场景1：从头开始**
1. 点击"步骤0 - 初始化"
2. 上传图片，获取Pet ID
3. 返回步骤选择器
4. 填写Pet ID
5. 继续后续步骤

### **场景2：跳过某些步骤**
假设你已经有了去背景的图片：
1. 在"上传图片"区域点击"上传去背景图片（步骤2用）"
2. 选择你的图片
3. 填写Pet ID和宠物信息
4. 点击"步骤2 - 生成基础坐姿图片"
5. 继续后续步骤

### **场景3：重新执行某一步**
假设步骤3失败了：
1. 填写Pet ID
2. 点击"步骤3 - 生成初始过渡视频"
3. 重新执行

### **场景4：只执行特定步骤**
假设你只想转换GIF：
1. 填写Pet ID
2. 点击"步骤6 - 转换为GIF"
3. 执行转换

---

## ⚠️ **注意事项**

### **1. Pet ID从哪里获取？**
- 在**步骤0（初始化）**完成后会生成
- 格式：`pet_1234567890`
- 可以在后端日志或前端提示中找到

### **2. 跳过步骤会怎样？**
- 如果跳过了前置步骤，后续步骤可能会失败
- 例如：没有执行步骤2，直接执行步骤3会失败（因为缺少坐姿图片）

### **3. 建议的执行顺序**
```
步骤0 → 步骤1 → 步骤2 → 步骤3 → 步骤4 → 步骤5 → 步骤6
```

但你可以根据需要跳过或重复某些步骤。

## 🆚 **对比：分步模式 vs 步骤选择器**

| 特性 | 分步模式 | 步骤选择器 |
|------|---------|-----------|
| **流程** | 必须按顺序 | 可以跳转 |
| **适合** | 第一次使用 | 熟悉流程后 |
| **灵活性** | ❌ 低 | ✅ 高 |
| **易用性** | ✅ 高（引导式） | ⚠️ 中（需要理解流程） |
| **重新执行** | ❌ 需要从头开始 | ✅ 可以直接跳到某步 |

## 💡 **最佳实践**

1. **第一次使用**：建议使用"分步模式"，了解完整流程
2. **熟悉后**：使用"步骤选择器"，提高效率
3. **调试时**：使用"步骤选择器"，快速定位问题
4. **重新执行**：使用"步骤选择器"，无需从头开始

## 🎉 **总结**

**步骤选择器**让你可以：
- ✅ 直接跳转到任意步骤
- ✅ 跳过不需要的步骤
- ✅ 重新执行失败的步骤
- ✅ 提高工作效率

**分步模式**让你可以：
- ✅ 按顺序执行所有步骤
- ✅ 有清晰的引导
- ✅ 适合第一次使用

**两种模式互补，根据需要选择！** 🚀
